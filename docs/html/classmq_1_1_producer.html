Detailed Description

template<typename MessageType, enabled< MessageType >>
class mq::Producer< MessageType, >

The producer class is used to send messages to various receivers objects.

Instances of this class cannot be copy-constructed or copy-assigned, but can be moved. Any number of listeners can be attached to a single instance.
