Detailed Description

Header-only template C++17 library which implements a message queue.

Exposes two main class templates:

  • Receiver<MessageType>;
  • Producer<MessageType>.

Enumeration Type Documentation

◆ FullQueuePolicy

enum mq::FullQueuePolicy
strong

Enum class which describes the action to undertake if the queue is full.

  • DO_NOTHING actually does nothing, the new message is lost;
  • SUB_FRONT substitute the last message with the incoming message;
  • REMOVE_BACK pop the first message and add the incoming message to the head of the queue;
  • THROW throw a FullQueueException. The incoming message is lost.
